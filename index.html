Git Tutorial

  • git status: mostra onde você está
  • git add: adciona novos arquivos
  • git commit -m "escreva seu texto aqui": descrição do projeto
  • git clone: cria uma cópia dum projeto
  • cd (change directory): muda de pasta
  • git checkout: volta para a última versão do commit
  • ls (list): mostra tudo que tem dentro da pasta
  • pwd: exibe o caminho do lugar que você está
  • git push: empurra os arquivos para o git
  • git init: inicia o repositório
  • git branch: para criar um branch (espacinho separado para trabalhar em um projeto)
